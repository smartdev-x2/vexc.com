# Vexc.com

This is a web3 project template built with:
- **Svelte**
- **Vite**
- **Tailwind CSS**

## Project Overview

This repository serves as a starting point for modern web3 apps using a frontend stack focused on speed and developer experience.

## Tech Stack

- **Svelte** – modern, lightweight, and fast UI framework
- **Vite** – lightning-fast build tool
- **Tailwind CSS** – utility-first CSS for rapid UI development

## Getting Started

1. **Clone the repository**
   ```
   git clone https://github.com/smartdev-x2/vexc.com.git
   cd vexc.com
   ```

2. **Install dependencies**
   ```
   npm install
   ```

3. **Start development server**
   ```
   npm run dev
   ```

The project includes the configuration for Svelte, Vite, and Tailwind CSS. Start building your decentralized app (dApp) right away!

## Customization

Feel free to tweak the template, add smart contracts, integrate wallets, or connect to blockchain networks as your web3 needs grow.
